PYPIRATES CarWash

A modern Car Wash & Vehicle Care web application developed as part of the **Software Innovation Challenge (SIC) 2026**. The platform provides an easy and convenient way for customers to book car wash services online, schedule home pickup, track service progress, and manage bookings through a simple dashboard.

👥 Team Details

**Team Name:** Pypirates

📌 Project Overview

PYPIRATES CarWash is designed to simplify the process of booking and managing vehicle cleaning services. Customers can choose services, schedule appointments, request home pickup, and monitor the status of their vehicle, while administrators can efficiently manage bookings and services.

Features

- 🚗 Online Car Wash Booking
- 📅 Schedule Service Appointments
- 🏠 Home Pickup & Delivery
- 📍 Service Status Tracking
- 👤 Customer Dashboard
- 🛠️ Admin Dashboard
- 📱 Responsive User Interface

## 🛠️ Technologies Used

🚀 Installation

Clone the repository:

```bash
git clone https://github.com/hashinikannan/Pypirates-carwash.git
```

Navigate to the project folder:

```bash
cd Pypirates-carwash
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
